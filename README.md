# How to Build
### On Linux
- make sure `g++` is installed
- Download the zip from repository
- extract it in desired directory
- `cd` into the folder containing the source code
- open a terminal and run:
```bash
  g++ main.cpp calculator.cpp -o calculator
```
- run it with `./calculator`, and enjoy a 'good' calculator!
